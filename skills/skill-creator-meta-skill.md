## Overview

Automates creation of AI skills by generating fully structured `.md` files.

## Features

- Generates skill metadata
- Includes detailed instructions
- Ready-to-use format

## Output Format

- Skill name
- Description
- Instruction steps

## Instructions

- Accept input goal
- Define role, task, process
- Output structured `.md` skill
