🎬 **The Pitch**  
You tell your AI “build me a fitness dashboard” → you get a **live, interactive preview** that instantly opens in the perfect Chrome window 📊.  

Changes? Live reload in sub-seconds ⚡.  
Love a version? It’s automatically saved to history.  
Want to experiment? Hit **“Save as New”** — your current preview stays safe.

**Artifact Preview v4.2** turns raw AI output into a delightful, version-controlled visual experience.

---

🔥 **Key Features**

- 🧠 Smart Auto-Detect → Portrait 📱 / Horizontal 📺 / Full 🖥️  
- ⚡ Sub-second live reload with Server-Sent Events  
- ✏️ Built-in editor with Code / Split / Preview tabs + Cmd+Shift+E  
- 📸 One-click Retina screenshots → Downloads + Preview app (Share ready)  
- 🕐 Recent Artifacts Dropdown with pinned “● Current — Live”  
- 📦 Persistent History — automatic timestamped backups  
- ✅ **“Save as New” button** — save variations safely without touching live preview  
- 🪟 Cross-platform launcher (native macOS feel)  
- 🎨 Modern dark UI with violet accents and smart auto-fit card  

**Idea → interactive prototype in ~3 seconds.** History included by default. 🔥

### 📥 **Installation** (One-line magic)

```bash
curl -fsSL https://raw.githubusercontent.com/ChuckSRQ/awesome-hermes-skills/main/skills/productivity/artifact-preview/install.sh | bash
```

After install:
```bash
cd ~/artifact-preview &amp;&amp; python3 server.py &amp;
bash ~/artifact-preview/open-chrome.sh
```

**Pro tip:** Add this alias:
```bash
alias ap='cd ~/artifact-preview &amp;&amp; python3 server.py &amp;'
```
