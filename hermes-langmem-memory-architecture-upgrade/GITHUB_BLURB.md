# GitHub repo blurb

## Repo name
hermes-langmem-memory-architecture-upgrade

## Short description
A shareable Hermes skill for upgrading the bundled LangMem provider into a structured, provenance-aware, locally testable memory architecture.

## About / long description
This repo packages a Hermes skill that upgrades the default LangMem setup from a basic memory hookup into a more deliberate local architecture with a typed profile lane, explicit provenance metadata, debounced background extraction, lightweight retrieval reranking, a separate episodic lane, a deterministic eval harness, and a reviewed procedural-memory workflow stub.

It is aimed at Hermes users who already have the bundled LangMem provider in their repo and want a clearer, safer, and more inspectable memory system without jumping straight to new infrastructure.
