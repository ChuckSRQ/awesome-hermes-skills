# Hermes Coding Skills for GitHub Copilot

This directory contains a GitHub Copilot plugin with coding and development skills from Hermes Agent.

## Installing (when `gh skill install` becomes available in gh CLI v2.90.0+)

```bash
gh skill install ChuckSRQ/awesome-hermes-skills:copilot/hermes-coding
```

Or install directly from this path:
```bash
copilot plugin install ChuckSRQ/awesome-hermes-skills:copilot/hermes-coding
```
